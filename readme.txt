For directions on how to use this R project script, including some information about system setup, please go to my website, https://nicolasmroberts.github.io/scripts 

This repository contains data for my 2021 Precambrian Research article, "Internal structure of the Paleoarchean Mt Edgar dome, Pilbara Craton Western Australia", with coauthor Basil Tikoff. A static, permanent version of this repository from March 2021 can be found at open science foundation:  https://osf.io/4je32
